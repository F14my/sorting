# Лабораторная работа №3

## Алгоритмический мини-пакет(структуры данных и сортировки)

### Описание

Проект содержит набор алгоритмов сортировки (bubble sort, quick sort, count sort, radix sort, insertion sort, bucket
sort, heap sort); генераторы тестовых данных, инструменты для измерения времени выполнения и интерактивный
CLI-интерфейс.

### Функциональность

#### Реализованы алгоритмы сортировки:

- Bubble sort
- Quick sort
- Heap Sort
- Counting Sort
- Radix Sort
- Bucket Sort (для вещественных чисел)
- Insertion Sort

#### Генераторы массивов

- ```rand_int_array``` - случайные целые числа
- ```rand_float_array``` - случайные вещественные числа
- ```nearly_sorted``` - почти отсортированный массив
- ```many_duplicates``` - массив с большим числом повторов
- ```reverse_sorted``` - массив, отсортированный по убыванию

#### Бенчмарки

В проекте был реализован модуль benchmark.py:

- timeit_once - измерение одного прогона функции(сортировки)
- benchmark_sorts - запуск бенчмарка для измерения времени сортировки

#### Интерактивный CLI

Примеры команд:

- show config
- set algo quick
- set size 10000
- set type nearly
- run sort
- run bench
- quit

Запуск бенчмарка:

```run bench```

Результаты записываются в benchmark.md

#### Поддержка ключей и компаратора

Для сортировок, основанных на сравнении(bubble_sort, quick_sort, heap_sort, insertion sort) реализована поддержка
параметров ```key``` и ```cmp```:
- ```key``` - функция, возвращающая значение, по которому производится сортировка элементов.
- ```cmp``` - пользовательская функция сравнения двух элементов, определяющая их порядок.
Если параметры не указаны, используется стандартное сравнение элементов.

### Тестирование

Проект использует pytest. Тестируются:

- Корректность сортировок
- Факториал и числа Фибоначчи
- Структуры данных(Стек для linked list)
- Ключи и компаратор

#### Запуск тестов:

```python -m pytest```

### Установка и запуск

1. Склонируйте репозиторий и перейдите в директорию проекта.
2. Создайте виртуальное окружение (например, через python -m venv .venv) и активируйте его.
3. Установить зависимости
   ```pip install -r requirements.txt```
4. Запустите оболочку:
   ```python -m src.main```
